# SysInfo
SysInfo is simple software to show information about your PC.

This software can display:

- CPU info

- Memory info

- GPU info

- Disk info

- Network info

- System Info(Version, System, etc..)

how to run:
 -     git clone https://github.com/GiltiusDev/SysInfo

 -     pip install requirements.txt

 -     python main.py
